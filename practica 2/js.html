JavaScript

Lenguaje de programacion ligero interpretado y basado en prototipos, esencial para crea sitios web interactivos y dinamicos. Permite añadir funcionalidades como animaciones, validación de formularios y actualizaciones de contenido en tiempo real. Se ejecuta principalmente en el navegador del cliente, aunque también se utiliza en el servidor con Node.js.
Características principales de JavaScript:
Interactividad: Dota de vida a las páginas web (el HTML es el esqueleto, CSS la apariencia y JS el comportamiento).
Multiparadigma: Soporta programación orientada a objetos, imperativa y funcional.
Dinámico: No requiere definición anticipada de tipos de datos.
Uso Generalizado: Aproximadamente el 88% de las páginas web utilizan JavaScript.
Ecosistema: Cuenta con numerosas librerías y frameworks (React, Angular, Vue) que amplían sus capacidades.
Ejemplos de uso:
Manipulación del DOM: Cambiar el contenido HTML, estilos CSS o estructura de la página.
Validación de formularios: Comprobar datos antes de enviarlos al servidor.
Llamadas asíncronas (AJAX/Fetch): Cargar datos sin recargar la página.
Aplicaciones web (SPA): Creación de aplicaciones complejas tipo YouTube o Facebook.
